# MX Bikes Controller Overlay (Local server for use in videos and livestreaming)

### Instructions:

Run the .exe file then add localhost:3000 as a browser source in OBS Studio.

## Xbox One Controller:

Clutch: LB

Front Brake: LT

Steering: L Joystick

Rear Brake: A

Lean: R Joystick

Dab: RB

## To Build Executable:
```
pkg . --targets node16-win-x64
```
